Getting Start

In this news blog project, Node.js with MongoDB CRUD allows you to easily manage blog content. You can:

  • Create new blog posts,
  • Read and display posts
  • Update existing posts, and
  • Delete old posts.

This basic CRUD setup helps you manage your blog's content easily and efficiently.

Requirement

  • Node.js installed on your system
  • MongoDB and MongoDB Compass installed locally.

Installation

Please download the project to a location of your choice on your computer and extract the contents of the downloaded ZIP file. Afterward, follow the instructions provided to complete the installation process :

  1. Open Project in a Code Editor.
  2. Install node dependencies run command in terminal : npm install
  3. Open MongoDB Connection file in PROJECT_FOLDER/config/mongoose-connection.js
    If you wish to change the database name, simply replace 'news' with your desired database name, or leave it as if no changes are required.

  4. Execute the following command in the terminal to start the project : npm start
  5. Import All MongoDB Collections using Compass Software from PROJECT_FOLDER/collections

  6. Proceed to : http://localhost:3000
  7. For Admin Panel : http://localhost:3000/admin
